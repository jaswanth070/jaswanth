Web Technologies Used

 This is a completely Django-based project which is built from scratch. This website was designed using the various services provided by the Django framework.

 This site is developed as part of a project for educational and knowledge purposes and is not meant for commercial.

 While coming to the front end, it was designed using the eminent front-end languages HTML, CSS, and Js. The pages of this website are designed by various inspiring websites on the internet.

 

 On the other hand, the backend was given pure python based codes as it uses a python framework. The website runs on the PostgresSQL database which is homed by Heroku - a free deployment platform.
